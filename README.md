# A-Meteo Landing

Локальная версия сайта `a-meteo`, подготовленная для запуска в Docker.

## Запуск

Самый удобный способ:

```powershell
.\run-docker.ps1
```

Скрипт сам:

- соберет Docker-образ;
- запустит контейнер;
- выберет свободный порт;
- выведет точный адрес сайта в консоль.

Обычно сайт будет доступен по адресу вида:

```text
http://localhost:8080
```

Если `8080` занят, скрипт поднимет сайт на следующем свободном порту.

## Ручной запуск

Если нужно собрать и запустить вручную:

```powershell
docker build -t a-meteo-local .
docker run --rm -p 8080:80 a-meteo-local
```
